Webseite Aufgabenstellung

Aufgabenbeschreibung

In dieser Aufgabe wird dir eine Seite mit fast nur HTML zur verfügung gestellt und du solltest die CSS datei editieren um die Webseite nach Anleitung zu verbessern.

Einschränkungen

  • KI darf verwendet werden für fragestellungen darf aber nicht für dich die Aufgabe lösen.
  • Dem Betreuenden Lernenden dürfen Fragen gestellt werden aber das Ziel ist Selbstsändige Problemlösung.

Aufgabe

Basis

  1. Der Titel sollte eine Hintergrundfarbe erhalten und Zentriert werden.
  2. Die Gesamte Anleitung sollte eine Abgerundete Umrandung erhalten
  3. Die Codeblocks sollten einen hellgrauen Hintergrund erhalten
  4. Die Beispiele sollten eine Eckige Umrandung erhalten
  5. Ihr solltet ein Eigenes Bild einfügen

Vortgeschritten

  • Ein Inhaltsverzeichniss von 0 auf erstellen mit logisch gesetzten springpunkten.

Bonus

Falls ihr das Alles erreich habt, könnt ihr mal versuchen einen Knopf mit java script zu machen der einen Alert gibt. Die Lernenden Helfen euch gerne dabei
