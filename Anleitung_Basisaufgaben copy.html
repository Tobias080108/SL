Merkpunkte Python

Kommentar :

Variable :

*xyz* oder *xyz* = *3*

Schleife :

for i in range(*Wiederholungen*):
*Code ist eingerückt*

Logikabfrage :

if *x == y*:
*Code ist eingerückt*

Funktionen :

def *Funktionsname*(*Wert1*, *Wert2*):
*Code ist eingerückt*

Funktion aufrufen :

*Funktionsname*(*Wert1*, *Wert2*)

(Angaben in ** sind Platzhalter für eigene Namen und Werte
